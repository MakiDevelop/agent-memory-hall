# ACA for CrewAI — Integration Guide

CrewAI natively supports MCP (Model Context Protocol) clients. Since AMH already exposes an MCP server, CrewAI agents can use ACA governance out of the box.

## Quick Start

```python
from crewai import Agent, Task, Crew
from crewai.mcp import MCPClient, StdioTransport

# Connect to AMH MCP server
mcp_client = MCPClient(
    transport=StdioTransport(
        command="npx",
        args=["agent-memory-hall", "serve", "--store", "sqlite"]
    )
)

# Agent with ACA-governed memory
researcher = Agent(
    role="Research Analyst",
    goal="Analyze data with full governance audit trail",
    mcp_clients=[mcp_client],
)
```

## What CrewAI Gets from ACA

### 1. Source Tier Tracking
Every memory written by CrewAI agents is tagged with `source_tier`:
- `raw_source` — data fetched from external sources
- `llm_derived` — conclusions generated by agents
- `human_confirmed` — facts verified by human review

### 2. Provenance Chain
When Agent A's output becomes Agent B's input, the provenance chain tracks:
```
CEO Summary (Agent C, llm_derived)
  ↑ synthesized from
Research Report (Agent B, llm_derived)
  ↑ derived from
Source Document (Agent A, raw_source)
```

### 3. Namespace Isolation
Each crew can run in its own namespace:
```python
# Each crew gets isolated memory
crew1_mcp = MCPClient(
    transport=StdioTransport(
        command="npx",
        args=["agent-memory-hall", "serve", "--caller-ns", "crew:research"]
    )
)
```

### 4. Audit Trail
Every memory read/write/revoke is recorded as an immutable audit event.

## Available MCP Tools

| Tool | Description |
|------|-------------|
| `amh_write` | Write a memory with governance metadata |
| `amh_read` | Read memories (namespace-isolated) |
| `amh_query` | Search memories by type, agent, status |
| `amh_revoke` | Revoke a memory (with audit trail) |
| `amh_tier_upgrade` | Upgrade trust tier (requires trust proof) |
| `amh_audit` | Get audit log for a memory |

## Task Provenance Pattern

```python
research_task = Task(
    description="Research topic X and store findings in ACA memory",
    agent=researcher,
    expected_output="Research findings stored with provenance",
)

analysis_task = Task(
    description="Analyze research findings from ACA memory, cite source memory IDs",
    agent=analyst,
    expected_output="Analysis with provenance chain linking to source memories",
)

crew = Crew(
    agents=[researcher, analyst],
    tasks=[research_task, analysis_task],
    mcp_clients=[mcp_client],
)
```

## Remote Connection (HTTP Transport)

```python
from crewai.mcp import MCPClient, HTTPTransport

mcp_client = MCPClient(
    transport=HTTPTransport(
        url="http://localhost:9100/mcp",
        headers={"Authorization": "Bearer YOUR_TOKEN"}
    )
)
```
